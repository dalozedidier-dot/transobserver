# Affichage GitHub : message “Uh oh! There was an error while loading”

Ce message peut provenir de l’interface GitHub (rendu/chargement), sans lien direct avec le contenu du dépôt.

## Contournements immédiats
- Recharger la page (plusieurs fois), changer de navigateur, désactiver temporairement les extensions.
- Accéder aux fichiers via l’URL *raw* (exemple ci-dessous) ou cloner le dépôt en local.

### Modèle d’URL raw
https://raw.githubusercontent.com/<owner>/<repo>/<branch>/README.md

## Escalade
Si le problème persiste durablement, ouvrir un ticket côté support GitHub (catégorie : “Repository view / rendering error”).
