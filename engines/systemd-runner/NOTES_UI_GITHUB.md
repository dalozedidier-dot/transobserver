# NOTES_UI_GITHUB.md

Le message GitHub : “Uh oh! There was an error while loading. Please reload this page.”
est un dysfonctionnement d’interface côté GitHub (rendu du listing / cache).

Contournements :
- recharger (hard reload),
- navigation privée / autre navigateur,
- utiliser les liens “Raw” (fichiers) et “Download ZIP” (repo),
- éviter les pushes massifs en rafales si possible (réduction des invalidations de cache).
